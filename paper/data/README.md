# Paper Data

This directory is for paper-specific data snapshots and provenance notes.

Rules:

- Every copied or digitized dataset needs source, DOI or URL, figure number, units, and digitization method.
- Generated data should record the command and commit hash.
- Do not store private data or personal contact details.
- Do not store third-party PDFs here.
