# Hysteresis Measurement Workflows: FORC + Literature Overlay

This document describes **text-first** measurement workflows in Module 1 and clearly labels outputs as:

- **Simulated**: Generated by the FeCIM model (Preisach or L-K)
- **Measured/Literature**: Imported from external data files

## 1) FORC workflow (First-Order Reversal Curves)

Use FORC mode to probe switching distribution and path dependence.

### Steps
1. Select material and physics engine.
2. Run reversal sweeps across multiple reversal fields.
3. Export data for offline analysis if needed.

### Output labeling
- `Simulated FORC` = generated from model state trajectory
- `Measured FORC` = lab-acquired FORC data (if externally imported)

Always report units in text:
- `E` in V/m (or MV/cm for readability)
- `P` in C/m² (or µC/cm² for readability)

## 2) Literature overlay workflow

Module 1 includes a literature overlay panel to compare model output against imported references.

### Supported inputs
- **CSV** with columns: `E_MV_cm,P_uC_cm2`
- **JSON** schema:

```json
{
  "source": "Park et al. 2015",
  "material": "HZO",
  "E_V_m": [...],
  "P_C_m2": [...]
}
```

### What the panel shows
- `Literature: <citation>, N=<points>`
- Fit metrics (computed after interpolation to measured E points):
  - RMSE (µC/cm²)
  - MAE (µC/cm²)
  - Max error (µC/cm²)
- Point-by-point table:
  - `E`
  - `P_sim` (**Simulated**)
  - `P_lit` (**Measured/Literature**)
  - `error = P_sim - P_lit`

## 3) Profile persistence

Overlay profiles can be saved/loaded as JSON and include source citation metadata.

Recommended fields:
- `source`
- `material`
- `e_field_v_m`
- `polarization_c_m2`
- `units`

## 4) Reporting template (text only)

- `Source: Park et al. 2015 (Measured/Literature)`
- `Simulation: Preisach model @ 300 K`
- `RMSE = 1.24 µC/cm², MAE = 0.98 µC/cm², MaxErr = 2.91 µC/cm², N = 48`
- `Interpretation: simulated loop reproduces measured coercive region within ±5% band`

This keeps simulated vs measured provenance explicit for accessibility and reproducibility.
