## Intuition

This module is the library card for the project.
It helps you find the right page quickly, shows you where you are,
and highlights glossary terms so you can keep reading without getting stuck.

## Key Analogies

- A map with landmarks and a "You are here" marker.
- A smart index that points you to the exact chapter you need.

## What the Simulator Simplifies

- Search ranking is approximate (TF-IDF, not semantic search).
- Only the curriculum folder is indexed by default.
