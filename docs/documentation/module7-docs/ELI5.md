## Intuition

The documentation viewer is a guided library.
It shows you the curriculum in a predictable order and lets you jump quickly
between ELI5, PHYSICS, FEATURES, and TOOLS for the same module.

Search is the fast path when you already know what you want.

## Key Analogies

- A textbook with tabs for each chapter and section.
- A map with a legend, compass, and quick shortcuts.

## What The Simulator Simplifies

- It is a document viewer, not a full web browser.
- Search is optimized for this repo, not the entire internet.
- Rendering focuses on clarity, not pixel-perfect typography.
