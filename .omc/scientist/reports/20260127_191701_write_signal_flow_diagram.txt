                          KIRCHHOFF'S LAWS DURING WRITE

KVL (Voltage Loop):
┌──────────────────────────────────────────────────────────────────┐
│         V_dac (+1.5V from DAC/charge pump)                       │
│            │                                                      │
│            ▼                                                      │
│     R_row (2.5Ω metal) → V_drop = I × R = 150µA × 2.5Ω = 0.375mV│
│            │                                                      │
│            ▼                                                      │
│     R_FeFET (~10kΩ) → Most voltage drops here                   │
│            │                                                      │
│            ▼                                                      │
│     R_col (2.5Ω metal) → V_drop = 0.375mV                       │
│            │                                                      │
│            ▼                                                      │
│          GND                                                      │
│                                                                   │
│  Loop equation:                                                  │
│    V_dac = I(R_row + R_FeFET + R_col) + V_cell                   │
│    1.5V ≈ 150µA × 10kΩ + 0.75mV                                 │
│    1.5V ≈ 1.5V + 0.00075V                                       │
│                                                                   │
│  Conclusion: IR drop negligible (<0.05%) during single-cell write│
└──────────────────────────────────────────────────────────────────┘

KCL (Current Conservation):
┌──────────────────────────────────────────────────────────────────┐
│             I_row_in                                             │
│                │                                                  │
│                ▼                                                  │
│         ┌─────────────┐                                          │
│         │  Cell (2,3) │  I_cell = 150 µA (during write)         │
│         └──────┬──────┘                                          │
│                │                                                  │
│                ▼                                                  │
│             I_col_out                                            │
│                                                                   │
│  KCL at cell node:                                               │
│    I_row_in = I_cell = I_col_out = 150 µA                       │
│                                                                   │
│  All other cells:                                                │
│    I_other = 0 (high impedance, not being written)              │
│                                                                   │
│  Conclusion: Simple current conservation during WRITE            │
│              (contrast with COMPUTE where all cells conduct)     │
└──────────────────────────────────────────────────────────────────┘
