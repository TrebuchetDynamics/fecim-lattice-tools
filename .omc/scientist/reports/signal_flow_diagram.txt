OPERATION 3: COMPUTE (4×4 Matrix-Vector Multiply)

Step 1: Input Vector → DAC Array (4 parallel DACs, 10ns)
  ┌──────────────────────────────────────────┐
  │  [1.0, 0.8, 0.6, 0.4] (digital)          │
  │         ↓  ↓  ↓  ↓                        │
  │    ┌───┬───┬───┬───┐                     │
  │    │DAC│DAC│DAC│DAC│ (5-bit each)        │
  │    └─┬─┴─┬─┴─┬─┴─┬─┘                     │
  │      V₀  V₁  V₂  V₃                      │
  └──────────────────────────────────────────┘

Step 2: Crossbar Array Computation (ALL 16 cells simultaneously, 5ns)
  ┌──────────────────────────────────────────┐
  │         V₀   V₁   V₂   V₃   (columns)    │
  │          │    │    │    │                 │
  │    WL₀ ──●────●────●────●───→ I₀         │
  │          │    │    │    │                 │
  │    WL₁ ──●────●────●────●───→ I₁         │  Physics:
  │          │    │    │    │                 │  • Each cell: I = G × V (Ohm's Law)
  │    WL₂ ──●────●────●────●───→ I₂         │  • Each row: ΣI (Kirchhoff's Law)
  │          │    │    │    │                 │
  │    WL₃ ──●────●────●────●───→ I₃         │  Result: Dot product in O(1) time!
  └──────────────────────────────────────────┘

  ┌──────────────────────────────────────────┐
  │  I₀=140.7µA → TIA → V=1.407V             │
  │  I₁=147.5µA → TIA → V=1.475V             │
  │  I₂=122.3µA → TIA → V=1.223V             │
  │  I₃=116.8µA → TIA → V=1.168V             │
  └──────────────────────────────────────────┘

Step 4: Voltage-to-Digital Conversion (4 parallel ADCs, 50ns)
  ┌──────────────────────────────────────────┐
  │  V → ADC → Digital Output Vector         │
  │  [1.407, 1.475, 1.223, 1.168]            │
  └──────────────────────────────────────────┘

PHYSICS EQUATIONS SUMMARY
═══════════════════════════════════════════════════════════════════════════════

1. OHM'S LAW (Analog Multiplication):
   ┌────────────────────────────────────┐
   │  I = G × V                         │
   │                                    │
   │  Example:                          │
   │  G = 52.2 µS (Level 15)            │
   │  V = 0.5 V (input)                 │
   │  I = 26.1 µA (output)              │
   │                                    │
   │  ✓ No ALU needed                   │
   │  ✓ Physics does the math           │
   │  ✓ Latency: ~1 ns (RC delay)       │
   └────────────────────────────────────┘

2. KIRCHHOFF'S CURRENT LAW (Analog Accumulation):
   ┌────────────────────────────────────┐
   │  I_row = Σ(G_ij × V_j)             │
   │                                    │
   │  Example (4 inputs):               │
   │  I = 18.1µA + 17.6µA + 15.7µA      │
   │      + 55.4µA                      │
   │    = 106.7 µA                      │
   │                                    │
   │  ✓ No adder circuit                │
   │  ✓ Physical wire summation         │
   │  ✓ Dot product in O(1) time        │
   └────────────────────────────────────┘

3. QUANTIZATION (30 Discrete Levels):
   ┌────────────────────────────────────┐
   │  G_i = G_min + (i/(N-1))×ΔG        │
   │                                    │
   │  i = 0 to 29                       │
   │  G_min = 1 µS                      │
   │  G_max = 100 µS                    │
   │  Precision: log₂(30) = 4.91 bits   │
   │                                    │
   │  Quantization error: <2% for MVM   │
   └────────────────────────────────────┘

KEY INSIGHTS
═══════════════════════════════════════════════════════════════════════════════

✓ WRITE: 150 ns, 0.13 pJ → Programs ferroelectric polarization state
✓ READ: 61 ns, 0.51 pJ → Non-destructive current sensing
✓ COMPUTE: 75 ns, 2.4 pJ for 16 MACs → 67× more efficient than GPU

WHY SO EFFICIENT?
• Weights stored IN the compute array (no DRAM access!)
• Physics performs multiplication (I=G×V) and accumulation (ΣI) simultaneously
• All cells operate in parallel → O(1) time for N² MACs

LIMITATIONS:
• Quantization: 30 levels → ~2% error (vs. 64-bit float)
• IR drop: Voltage attenuation in large arrays (>64×64)
• Sneak paths: 5-20% error in passive (0T1R) arrays
• Device variation: ±3-5% conductance spread
