READ OPERATION SIGNAL FLOW: MODULE 2 + MODULE 4

┌─────────────────────────────────────────────────────────────────────────────┐
│                         FERROELECTRIC CELL ARRAY                            │
│                              (Module 2)                                     │
└─────────────────────────────────────────────────────────────────────────────┘

WORD LINES (Rows) →  ║  ║  ║  ║  ║  ║  ║  ║   ← Input voltages V[i]
                     ═══════════════════════
         BIT LINE 0  ║⚡⚡⚡⚡⚡⚡⚡⚡║  Each ⚡ = ferroelectric cell
         (Column 0)  ║⚡⚡⚡⚡⚡⚡⚡⚡║  Conductance G[i,j] ∈ [10µS, 100µS]
                     ║⚡⚡⚡⚡⚡⚡⚡⚡║  30 discrete levels
         BIT LINE 1  ║⚡⚡⚡⚡⚡⚡⚡⚡║
         (Column 1)  ║⚡⚡⚡⚡⚡⚡⚡⚡║  Physics at each cell:
                     ║⚡⚡⚡⚡⚡⚡⚡⚡║  I[i,j] = G[i,j] × V[j]  ← Ohm's Law
                     ═══════════════════════
                       ↓   ↓   ↓   ↓   ↓
                     I₀  I₁  I₂  I₃  I₄ ...  ← Column currents

┌─────────────────────────────────────────────────────────────────────────────┐
│         KIRCHHOFF'S CURRENT LAW SUMMATION (Passive, Module 2)              │
└─────────────────────────────────────────────────────────────────────────────┘

    For each column j:
    
    I_out[j] = I[0,j] + I[1,j] + I[2,j] + ... + I[N-1,j]
             = G[0,j]×V[0] + G[1,j]×V[1] + ... + G[N-1,j]×V[N-1]
             = Σᵢ G[i,j] × V[i]  ← Matrix-Vector Multiply
    
    Physical implementation: Currents merge at bit line node (no active circuit)

┌─────────────────────────────────────────────────────────────────────────────┐
│                NON-IDEALITIES (Module 2 Corrections)                        │
└─────────────────────────────────────────────────────────────────────────────┘

    IR DROP (resistive voltage loss):
    ┌─────────────────────────────────────────────────┐
    │  V_eff[i,j] = V_ideal[i,j] - ΔV_wl - ΔV_bl    │
    │                                                 │
    │  ΔV_wl = j × R_wl × I_row   (cumulative drop)  │
    │  ΔV_bl = i × R_bl × I_col   (cumulative rise)  │
    │                                                 │
    │  Worst cell: [Rows-1, Cols-1] (max path R)     │
    │  Typical: 3-10% voltage reduction              │
    └─────────────────────────────────────────────────┘

    SNEAK PATHS (parasitic currents):
    ┌─────────────────────────────────────────────────┐
    │  0T1R (passive): Sneak ≈ Signal (ratio ~1.0)   │
    │  1T1R (active):  Sneak ≈ 0.001 × Signal        │
    │                                                 │
    │  Three-cell loop: G_sneak = 1/(1/G₁+1/G₂+1/G₃) │
    └─────────────────────────────────────────────────┘

    DEVICE VARIATION & DRIFT:
    ┌─────────────────────────────────────────────────┐
    │  G_eff = G_nom × (1 + noise) × drift(t)        │
    │  noise ~ N(0, σ²) ← Gaussian                   │
    │  drift(t) = (t/t₀)^(-ν) ← Power law            │
    └─────────────────────────────────────────────────┘

    Output: I_out[j] ∈ [10µA, 100µA] (physical current range)

        ↓ ↓ ↓ ↓ ↓  (One current per output column)

┌─────────────────────────────────────────────────────────────────────────────┐
│              TRANSIMPEDANCE AMPLIFIER (TIA) - Module 4                      │
└─────────────────────────────────────────────────────────────────────────────┘

    Circuit:  ┌────── R_feedback = 10 kΩ ──────┐
              │                                 │
    I_in ────┤─         ___                    │
              │        /   \                    │
              └───────| OpAmp|───────────────────── V_out
                       \___/         │
                                     │
                                   ┴ GND

    Transfer Function:
    ┌─────────────────────────────────────────────────┐
    │  V_out = I_in × R_feedback + V_offset          │
    │        = I_in × 10kΩ + 5mV                     │
    │                                                 │
    │  Example (I_in = 50µA):                        │
    │  V_out = 50µA × 10kΩ = 500mV                   │
    └─────────────────────────────────────────────────┘

    Performance:
    ┌─────────────────────────────────────────────────┐
    │  Bandwidth:      100 MHz (-3dB)                │
    │  Settling time:  ~11 ns (0.1% accuracy)        │
    │  Input noise:    1 pA/√Hz                      │
    │  RMS noise:      100 µV (at 100 MHz BW)        │
    │  Dynamic range:  140 dB                        │
    │  SNR (max):      80 dB (1V signal)             │
    └─────────────────────────────────────────────────┘

    Output: V_tia ∈ [0V, 1V] (analog voltage)

        ↓ ↓ ↓ ↓ ↓

┌─────────────────────────────────────────────────────────────────────────────┐
│        ANALOG-TO-DIGITAL CONVERTER (ADC) - Module 4                         │
└─────────────────────────────────────────────────────────────────────────────┘

    Architecture: SAR (Successive Approximation Register)
    
    Conversion Process (5 cycles):
    ┌─────────────────────────────────────────────────┐
    │  Cycle 1: Test MSB (bit 4) → V > Vref/2?      │
    │  Cycle 2: Test bit 3       → V > Vref/4?      │
    │  Cycle 3: Test bit 2       → V > Vref/8?      │
    │  Cycle 4: Test bit 1       → V > Vref/16?     │
    │  Cycle 5: Test LSB (bit 0) → V > Vref/32?     │
    │                                                 │
    │  Each cycle: ~10 ns → Total: 50 ns             │
    └─────────────────────────────────────────────────┘

    Quantization:
    ┌─────────────────────────────────────────────────┐
    │  Levels:         32 (5 bits, use 30 for FeCIM)│
    │  LSB size:       1V / 31 = 32.26 mV            │
    │  Code = round(V_in / LSB)                      │
    │                                                 │
    │  Example (V_in = 500mV):                       │
    │  Code = round(500mV / 32.26mV) = 15            │
    └─────────────────────────────────────────────────┘

    Nonlinearity:
    ┌─────────────────────────────────────────────────┐
    │  INL: 0.5 LSB (integral nonlinearity)          │
    │  DNL: 0.25 LSB (differential nonlinearity)     │
    │  ENOB: ~4.77 bits (effective resolution)       │
    │  SQNR: 31.9 dB (signal-to-quantization noise)  │
    └─────────────────────────────────────────────────┘

    Output: Digital code ∈ [0, 29] (integer)

        ↓ ↓ ↓ ↓ ↓

================================================================================
KIRCHHOFF'S LAWS SUMMARY
================================================================================

    CURRENT LAW (KCL):
        At each bit line node: Σ I_in = Σ I_out
        
        Enables passive analog computation (no active adder needed)
        Implementation: Physical wire junction at bit line
    
    VOLTAGE LAW (KVL):
        Around any loop: Σ V = 0
        
        V_driver - Σ(I×R_wl) - V_cell - Σ(I×R_bl) - V_ground = 0
        
        Explains IR drop: Voltage consumed by wire resistance
        Worse at array corners (longest current paths)

================================================================================
MODULE INTEGRATION
================================================================================

    Module 2: Produces CURRENTS (physical quantities)
              ├─ I_out[j] = Σᵢ G[i,j] × V[i]  ← KCL
              ├─ Range: 10µA - 100µA
              └─ Includes: IR drop, sneak, drift, noise

    Module 4: Senses CURRENTS → Produces CODES (digital outputs)
              ├─ TIA:  I → V    (V = I × 10kΩ)
              ├─ ADC:  V → Code (Code = round(V/LSB))
              └─ Range: 0 - 29 (30 discrete levels)

    NO DIRECT CODE INTERFACE (separate simulation domains)
    Integration is CONCEPTUAL (documentation + user workflow)
