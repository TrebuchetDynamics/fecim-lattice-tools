# Calibration Pack (Literaturized Targets)

This folder pins a minimum calibration pack used by CI-style tests.

## Targets

1. **P-E loop frequency sweep** (1 Hz, 1 kHz, 1 MHz)
2. **Switching time vs pulse amplitude** (3 amplitudes)
3. **Read margin vs array size** (32, 64, 128, 256)

## Sources and citations

- **HfO2/HZO ferroelectric loop behavior (baseline P-E context):**
  - M. H. Park *et al.*, “Ferroelectricity and Antiferroelectricity of Doped Thin HfO2-Based Films,” *Advanced Materials* **27** (11), 1811–1831 (2015). https://doi.org/10.1002/adma.201404531

- **Landau/LK-oriented HfO2 coefficient baseline used in this repository:**
  - R. Materlik, C. Künneth, A. Kersch, “The origin of ferroelectricity in Hf1−xZrxO2: a computational investigation and a surface energy model,” *Journal of Applied Physics* **117**, 134109 (2015). https://doi.org/10.1063/1.4916229

- **Nucleation-limited switching spread reference used for parameterization context:**
  - R. Guo *et al.*, “Nucleation-limited polarization switching in ferroelectric HfO2 capacitors,” *Applied Physics Letters* **112**, 262903 (2018). https://doi.org/10.1063/1.5038038

- **Heracles comparator paper used by sibling harness (for model-comparison context):**
  - arXiv:2410.07791, “Heracles” ferroelectric compact-model publication (2024). https://arxiv.org/abs/2410.07791

- **Array-level read-margin trend context (IR-drop / sneak-path scaling):**
  - Generic resistive-crossbar and FeCIM literature trend; this pack uses pinned synthetic points for regression until a single canonical open dataset is adopted.

## Notes

- Current targets are **pinned calibration anchors** for regression and are intentionally approximate.
